    Introduction

    Welcome to the Pasta API! You can use this API to access Pasta API endpoints, which can get information on various pasta types in our database (Source).

    The language binding is in JavaScript utilizing JQuery. You can view code examples in the dark area to the right.

    This API documentation page was created with Slate.

    Pastas

    Get All Pastas

    $.get('http://gpolyak.xyz/pasta_api/v1/all/', function(result) {
  //DO SOME WORK
});

    The above command returns JSON structured like this:

    [
  {
    "id": 2,
    "name": "Alphabet Pasta",
    "description": "This favorite kids’ shape makes any meal fun.",
    "uses": "Have a child that’s too picky to eat their veggies? Pair alphabet pasta with some vegetables or soup to win them over.",
    "classification": "Shape"
  },
  {
    "id": 3,
    "name": "Bucatini",
    "description": "This straw-like pasta is shaped like thick Spaghetti but is hollow in the center.",
    "uses": "Bucatini is the perfect choice for nearly any sauce, or it can be baked in casseroles or stirfried in dishes. Try it with different lean proteins and sauces for a change of pace.",
    "classification": "Shape"
  }
]

    This request retrieves all Pastas.

    HTTP Request

    GET http://gpolyak.xyz/pasta_api/v1/all/

    Query Parameters

    Parameter Default Description
    classification null If set to 'Noodle' then the result will only be noodles, 'Shape' returns only shapes, and null returns everything.

    Get a Specific Pasta

    var data = {id:2};
$.get('http://gpolyak.xyz/pasta_api/v1/all/', data, function(result) {
  //DO SOME WORK
});

    The above command returns JSON structured like this:

    {
  "id": 2,
  "name": "Alphabet Pasta",
   "description": "This favorite kids’ shape makes any meal fun.",
  "uses": "Have a child that’s too picky to eat their veggies? Pair alphabet pasta with some vegetables or soup to win them over.",
  "classification": "Shape"
}

    This request retrieves a specific Pasta.

    HTTP Request

    GET http://gpolyak.xyz/pasta_api/v1/all/

    URL Parameters

    Parameter Description
    id If set to a specific id, the entry with that id will be returned.

    Delete a Specific Pasta

    var data = {id:2, "delete":"true"};
$.get('http://gpolyak.xyz/pasta_api/v1/all/', data, function(result) {
  //DO SOME WORK
});

    The above command returns an HTTP response code of 200 after deletion.

    This request deletes a specific Pasta.

    HTTP Request

    GET http://gpolyak.xyz/pasta_api/v1/all/

    URL Parameters

    Parameter Description
    delete If set to true, the entry containing the provided id will be deleted.
    id If set to a specific id and the delete parameter is not provided, the entry with that id will be returned.

    Add a new Pasta to the database

    var data = {name:"someNoodle",
    description:"Example noodle description",
    uses:"Example noodle uses",
    classification: "Shape"};
$.post('http://gpolyak.xyz/pasta_api/v1/all/', data, function(result) {
  //DO SOME WORK
});

    The above command returns an HTTP response code of 200 after the addition.

    This request adds a new Pasta.

    HTTP Request

    POST http://gpolyak.xyz/pasta_api/v1/all/

    URL Parameters

    Parameter Description
    name Name of the Pasta
    description Description of the Pasta.
    uses Possible uses of the Pasta.
    classification Classification of the Pasta (Shape or Noodle).

    Edit an existing Pasta in the database

    var data = {id:2,
    name:"someNoodle",
    description:"Example noodle description",
    uses:"Example noodle uses",
    classification: "Shape"};
$.post('http://gpolyak.xyz/pasta_api/v1/all/', data, function(result) {
  //DO SOME WORK
});

    The above command returns an HTTP response code of 200 after the edit.

    This request edits an existing Pasta.

    HTTP Request

    POST http://gpolyak.xyz/pasta_api/v1/all/

    URL Parameters

    Parameter Description
    id ID of the Pasta intended to be edited
    name Name of the Pasta
    description Description of the Pasta.
    uses Possible uses of the Pasta.
    classification Classification of the Pasta (Shape or Noodle).

    See options

    var data = {options:"true"};
$.post('http://gpolyak.xyz/pasta_api/v1/all/', data, function(result) {
  //DO SOME WORK
});

    The above command returns an HTTP response code of 200 and provides the allowed request methods that this API can use.

    This request shows the request methods available.

    HTTP Request

    POST http://gpolyak.xyz/pasta_api/v1/all/

    URL Parameters

    Parameter Description
    options If set to true, the allowed request methods that this API can use is provided.

    Endpoints

    Here are the different Pasta endpoints publicly available.

    All

    http://gpolyak.xyz/pasta_api/v1/all/

    Noodles

    http://gpolyak.xyz/pasta_api/v1/noodles/

    Shapes

    http://gpolyak.xyz/pasta_api/v1/shapes/

    Errors

    The Pasta API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- Your request sucks.
    404 Not Found -- The specified pasta could not be found.
